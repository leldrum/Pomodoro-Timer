# Pomodoro Timer

Un chronomètre Pomodoro simple et réactif, développé avec HTML, CSS et JavaScript. Ce projet permet aux utilisateurs de gérer leur temps de travail et de pause en suivant la technique Pomodoro.

## Table des matières

## Caractéristiques

- Interface utilisateur intuitive
- Paramètres personnalisables pour les minutes de travail et de pause
- Affichage du temps restant avec un timer
- Modifications dynamiques de l'interface en fonction des actions de l'utilisateur
- Support pour les appareils mobiles et les ordinateurs

## Technologies

- HTML
- CSS
- JavaScript
- Font Awesome pour les icônes
